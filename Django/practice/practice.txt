what is Templates
- Templates stores html pages
- application level 
    - create Templates folder on application
    - when we are creating application level Templates
    - no need to change settings.py
- project level
    - create Templates folder on project
    - we need to set the path for the Templates in settings.py 
    - inside that serarch Templates
        - import os
        - "DIRS": [os.path.join(BASE_DIR,"templates")]


from django.shortcuts import render

static folder
- images
- css
- js

- created static file inside project 
- to locate static folder we need to add path inside the setting.py
add 
STATICFILES_DIRS = [os.path.join(BASE_DIR,"static")] 
inside setting.py

- to use static files we need to load inside the page
    - {% load static %}
- link css 
    <link rel="stylesheet" href="{% static 'css/styles.css' %}">
