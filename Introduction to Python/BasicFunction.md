


# **The `type()` Function**
This predefined function returns the data type of a variable.
```python
a = 30
print(type(a))  # Output: <class 'int'>
```

# **The `id()` Function**

The `id()` function is predefined in Python and provides the memory address where the variable's data is stored.

```python
a = 1001
print(a)
print(id(a))
```
